This is the API

busqueda

Busca comentarios a partir de unas fechas de inicio y fin, asi como una lista de temas y unas condiciones.

URL

https://virtserver.swaggerhub.com/Di3GO95/API_REST_RRSS/1.0.0/temas/search

HTTP Method

GET

Response Type

List

Parameters

  • Query: since String

    Fecha de inicio, a partir de cuando buscar los comentarios.

  • Query: until String

    Fecha de fin, hasta cuando buscar los comentarios.

  • Query: tema List(String)

    Temas sobre los que buscar

getComentario

Obtiene un comentario de un tema proporcionando el id del tema y del comentario

URL

https://virtserver.swaggerhub.com/Di3GO95/API_REST_RRSS/1.0.0/temas/{idTema}/{idComentario}

HTTP Method

GET

Response Type

Comentario

Parameters

  • Path: idTema String

    ID del tema sobre el que buscar el comentario

  • Path: idComentario String

    ID del comentario que se quiere buscar
